# smtp-server
计算机网络课程作业 #2 —— 实现简单的SMTP邮件服务器

可能还会有一些bug，有问题请多指教。

最终得分：97/100

## 编程环境

Visual Studio 2015 (for C++, MFC应用程序)，使用的类是`CAsyncSocket`。

- 编译选项最好使用Debug，否则可能会崩
- 服务器的IP只能使用本地IP，例如`127.0.0.1`

## 基本功能

- 接收从 Outlook 或 Foxmail 客户端发来的邮件
- 显示邮件中的各种格式的一张图片，可以是直接添加到正文的图片，也可以是来自附件的图片
- 显示邮件中的附件文件名及邮件正文内容
- 保存图片和其他所有附件到本地
- 显示交互日志
- 支持多个邮箱用户
- 可重复接收邮件

## 拓展功能

- 邮件内容中文显示
